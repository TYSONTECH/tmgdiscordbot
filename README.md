# YouTube Content Discord bot

**Python Code for a discord bot to notify servers added**  
_Made by Tyler Custine_

---

## Overview

It's a discord bot

---

## Features

- **Server Content Notifications**  
  Effortlessly notify the servers that you've added the discord bot to about new content from your YouTube Channel using the YouTube Data API v3

- **Direct Message Content Notifications**  
  Effortlessly direct message a list of users that you've specified in the python code about new content from your YouTube Channel using the YouTube Data API v3

- **Wake-on-LAN & Sleep-on-LAN**  
  Remotely wake or sleep devices to optimize power usage and accessibility.
